# Ice Games – Regional & Platform Sales Analysis

Proyecto de análisis de datos (Ice Games): ventas por región/plataforma, segmentación de usuarios y hallazgos accionables.

## Objetivo
Identificar patrones de ventas y comportamiento de usuarios por región y plataforma para apoyar decisiones de negocio.

## Dataset
Datos históricos de ventas y comportamiento de usuarios (plataforma, región, métricas de compra/uso).

## Herramientas
Python (pandas, numpy, matplotlib/seaborn), Jupyter Notebook.

## Qué se hizo
- Limpieza y preparación de datos
- Análisis exploratorio (EDA)
- Comparación por región y plataforma
- Segmentación de usuarios
- Conclusiones y recomendaciones

## Resultados (resumen)
- Hallazgos clave sobre regiones con mejor desempeño
- Diferencias relevantes entre plataformas
- Segmentos de usuarios con mayor valor

## Estructura del repositorio
- `notebooks/` – análisis en Jupyter
- `data/` – dataset (si se puede publicar)
- `reports/` – conclusiones o presentación (opcional)
